# AIDEN Product

Everything built by the AIDEN AI agent team lives here.

## Structure

```
product/          # Actual product code and assets
  code/           # Application code
  landing-pages/  # Landing page HTML/CSS
  assets/         # Images, videos, design files
  templates/      # Product templates (Notion, Gumroad, etc.)

research/         # Market research and analysis
  market/         # Market research reports
  competitors/    # Competitor analysis
  leads/          # Lead research and outreach lists

content/          # Social media and marketing content
  drafts/         # Pending approval
  approved/       # Approved, ready to post
  published/      # Already posted

reports/          # Status reports, reviews, retrospectives
decisions/        # Business decisions with reasoning
```

## How it works

The AIDEN agent team commits directly to this repo:
- **Blake (Builder)** commits code, landing pages, and assets
- **Maya (Marketing)** commits research reports and campaign plans
- **Sam (Sales)** commits lead research and outreach drafts
- **Parker (PM)** commits status reports
- **AIDEN (CEO)** commits decisions

Every commit is a real, verified action tracked in the dashboard.
